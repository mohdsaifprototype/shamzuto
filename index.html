Steps to initialize Git and publish (summary)

  1. Open your project in VS Code
    Open the folder that contains your project files in VS Code.
  2. Initialize a Git repository
    Use the Source Control panel or run git init in the terminal.
  3. Make first commit
    Stage files and commit using the UI or git add . and git commit -m "Initial commit".
  4. Create repository on GitHub
    Create a new repo on github.com and copy the remote URL.
  5. Push from VS Code
    Add remote and push: git remote add origin <url>, git push -u origin main.
  6. Verify on GitHub
    Open the repo page on GitHub to confirm files are published.
